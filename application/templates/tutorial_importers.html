How to import an inventory?

Right click on the first item of the desired inventory (base, ship, etc.) and select "Inspect". Scroll up a bit and right click on the "<div class="InventoryView__grid___XXXX">" element. Hover copy.

On Chrome: select "Copy Element"
On Firefox: select "Copy Inner HTML"

Paste the string in the field and submit.

You should get a printed table as follow:

MAT Ticker Amount

Example:

BHE 12
DW 134

Copy/paste the table in your favorite spreadsheet...

The same procedure applies to Production Lines, with the elements described on that page. Currently working for one production line at a time...
